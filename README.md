# Reading Tracker Application

## Description
A Spring Boot-based web app that helps users manage and track their reading progress.

## Technologies Used
Spring Boot, Spring Security, Thymeleaf, MySQL, Bootstrap, Maven.

## How to Run
1. Clone the repository.
2. Configure database in application.properties.
3. Run using IntelliJ IDEA or command:
   ```bash
   mvn spring-boot:run
